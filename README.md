### 🚑 AmbuHelp
**Emergency ambulance dispatch, with live GPS tracking**

A hackathon build for emergency response — patients can request an ambulance with zero login required, hospitals get a live dashboard of incoming patients with vitals, and dispatch runs over WebSockets so everyone sees the same real-time map.

- Built across **frontend + backend** with my team
- FastAPI backend, MongoDB, JWT auth for staff roles
- Live GPS tracking + ETA via OSRM routing
- Role-based views: dispatcher, driver, hospital, public emergency portal
- Rule-based hospital matching (manual or auto-assist mode)

`FastAPI` `MongoDB` `WebSocket` `JWT` `HTML/CSS` `TypeScript`

[**View repo →**](https://github.com/snehapradeepraj/ambuhelp)

### 🚇 KMRL Metro DSS
**Decision-support system for metro trainset scheduling**

A Django app built for a railway-operations hackathon problem statement — centralizing trainset availability, fitness certificates, branding-hour tracking, and maintenance priorities that teams were previously managing across spreadsheets.

- Built across **frontend + backend** with my team
- Django + PostgreSQL, REST endpoints via DRF
- Optimization layer for scheduling (`scikit-learn`, `PuLP`)
- Custom management commands for data import & model training
- Structured data models for trainsets, contracts & job cards

`Django` `PostgreSQL` `scikit-learn` `PuLP` `Python`

[**View repo →**](https://github.com/snehapradeepraj/kmrl-metro-dss)
